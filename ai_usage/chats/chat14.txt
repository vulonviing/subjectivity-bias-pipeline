 Verified facts (Rule 5)

 - Endpoint'ler: /v1/chat/completions, /v1/responses (+ embeddings, completions, moderations). Biz /v1/chat/completions kullanacağız (geniş tooling, structured outputs
 destekli).
 - Input: JSONL, max 50,000 request / 200 MB. Her satırda custom_id zorunlu ve unique.

 - completion_window: yalnızca "24h". Batch genellikle daha hızlı biter.
 - Output line order ≠ input line order → custom_id ile join şart.
 - Pricing: synchronous API'ye karşı %50 discount.
 - gpt-5.4-mini: Chat Completions + Responses API + Structured Outputs + Batch destekli (README'de zaten 2026-06-01 verify edildi).

     Pipeline'ın bir sonraki adımı: her cümle için silver / reference annotation üretmek (objective vs subjective_framing). Hedef → proxy etiketleri ile agreement analizi.

     Bu plan sadece infrastructure'ı kurar: OpenAI Batch API üzerinden submit/status/fetch akışı, structured outputs ile JSON dönüş, deterministic state tracking. Prompt tasarımı
      ve JSON response schema'sı kasıtlı olarak deferred — kullanıcı bunları ayrı bir adımda çalışacak. Bu yüzden script, gerçek prompt content olmadan submit'i reddedecek; ama
     status / fetch / parsing tarafı baştan sona implement edilecek.

     Veri ölçeği: 8,561 cümle = 1 batch (OpenAI limiti 50,000 request / 200MB — bizim için bol).

     Verified facts (Rule 5)

     OpenAI Batch API resmi spec'inden teyit (2026-06-03):
     - Endpoint'ler: /v1/chat/completions, /v1/responses (+ embeddings, completions, moderations). Biz /v1/chat/completions kullanacağız (geniş tooling, structured outputs
     destekli).
     - Input: JSONL, max 50,000 request / 200 MB. Her satırda custom_id zorunlu ve unique.
     - Batch object alanları: id, status ∈ {validating, in_progress, finalizing, completed, failed, expired, cancelled}, output_file_id, error_file_id, request_counts.
     - completion_window: yalnızca "24h". Batch genellikle daha hızlı biter.
     - Output line order ≠ input line order → custom_id ile join şart.
     - Pricing: synchronous API'ye karşı %50 discount.
     - gpt-5.4-mini: Chat Completions + Responses API + Structured Outputs + Batch destekli (README'de zaten 2026-06-01 verify edildi).

     Decisions (kullanıcı onaylı)

     1. Kapsam: Tam 8,561 cümle (sub-sample yok). Tek batch.
     2. Input: data/processed/proxy_predictions.jsonl (proxy_label/score zaten içeride → agreement analizinde join'siz çalışılır).
     3. Script yapısı: Tek dosya src/annotate_llm.py, üç subcommand: submit, status, fetch.
     4. Model pin: OPENAI_MODEL_SNAPSHOT (gpt-5.4-mini-2026-03-17); yoksa OPENAI_MODEL (gpt-5.4-mini) fallback.
     5. Prompt + response schema: DEFERRED — kullanıcı bu adımı ayrı çalışacak. submit subcommand'ı placeholder dosya tespit ederse loud-fail eder.

     submit subcommand:
     1. .env load (python-dotenv). Resolve model: snapshot > model > default. API key zorunlu.
     2. _load_prompt(path) → dosya içeriğini oku.
       - Placeholder check: dosya _PLACEHOLDER_MARKERS'tan birini içeriyorsa veya boşsa → SystemExit("Prompt is still a placeholder. Design prompts/llm_sentence_annotation.txt
     before submit.").
       - Beklenen format ilerideki adımda netleşecek; şimdilik tek string olarak okunur ve user message'a {sentence} substitution'ı için Python str.format ile geçilir. (Format
     detayı prompt step'te kesinleşecek; script KeyError: 'sentence' olursa anlamlı hata mesajı verir.)
     3. _load_schema(path) → JSON schema oku (Structured Outputs için response_format = {"type": "json_schema", "json_schema": {...}}). Schema dosyası placeholder ise yine fail.
     4. proxy_predictions.jsonl stream-read. Her row için:
     {
       "custom_id": "<sentence_id>",
       "method": "POST",
       "url": "/v1/chat/completions",
       "body": {
         "model": "<resolved-model>",
         "messages": [
           {"role": "system", "content": <system_part>},
           {"role": "user",   "content": <user_template>.format(sentence=row["text"])}
         ],
         "response_format": {"type": "json_schema", "json_schema": <schema>},
         "temperature": 0,
         "max_completion_tokens": 256,
         "seed": 42
       }
     }
     4. Modele gönderilen alanlar yalnızca cümle metni — outlet, URL, title, image gönderilmez (CLAUDE.md). Metadata yalnızca output join için saklı.
     5. Geçici JSONL'i data/processed/llm_batch_input.jsonl olarak yaz, sonra OpenAI files.create(file=..., purpose="batch") → input_file_id.
     6. batches.create(input_file_id, endpoint=BATCH_ENDPOINT, completion_window=COMPLETION_WINDOW, metadata={"step":"annotate_llm","n":N}) → batch_id.
     7. State file yaz:
     {
       "batch_id": "...",
       "input_file_id": "...",
       "model": "gpt-5.4-mini-2026-03-17",
       "endpoint": "/v1/chat/completions",
       "n_requests": 8561,
       "input_path": "data/processed/proxy_predictions.jsonl",
       "submitted_at": "2026-06-03T...Z"
     }
     8. --dry-run: hiçbir API çağrısı yok, sadece batch input JSONL'i üretip ilk 3 satırı stdout'a basar (prompt step'inde debug için faydalı).

     status subcommand:
     1. State file oku → batch_id.
     2. batches.retrieve(batch_id) → status, request_counts, output_file_id, error_file_id.
     3. Tablo formatında log; status completed ise "ready to fetch", failed/expired/cancelled ise loud-fail.

     fetch subcommand:
     1. State file oku → batch_id.
     2. batches.retrieve(batch_id) → status completed değilse SystemExit.
     3. files.content(output_file_id) → JSONL stream. Her satır parse:
     {"custom_id":"<sentence_id>", "response": {"status_code":200, "body": {...chat completion...}}, "error": null}
     4. Input proxy_predictions.jsonl tekrar oku (custom_id → original row map).
     5. Her output satırı için:
       - error non-null → llm_annotations.errors.jsonl'a ekle.
       - response.body.choices[0].message.content → JSON parse → {llm_label, llm_rationale, llm_confidence}.
       - Parse fail → errors file.
       - Output row = orijinal proxy_predictions row + llm_label, llm_rationale, llm_confidence, llm_model, llm_snapshot (= resolved model id), llm_finish_reason.
       - --keep-raw flag'i varsa llm_raw_response da eklenir (debug için).
     6. JSONL yaz data/processed/llm_annotations.jsonl. Her sentence_id için tek satır, custom_id ile join.
     7. Özet log: total / succeeded / failed / token usage breakdown / cost tahmini (response.body.usage toplamlarından, batch discount applied).

     Determinism:
     - temperature=0, seed=42 her request'te. (OpenAI seed best-effort, exact match garantisi yok ama logla.)

     Logging: logging.basicConfig(INFO, stderr). Submit'te N request, fetch'te per-row tqdm gerekmez (single read pass).

      141 +    """Download image URL, resize to max 1024px, encode as JPEG q85. Cache on disk."""                                                                          
      142 +    if cache_path.exists():                                                                                                                                     
      143 +        return cache_path.read_bytes()                                                                                                                          
      144 +                                                                                                                                                                
      145 +    user_agent = os.getenv("USER_AGENT", "media-bias-pipeline/1.0")                                                                                             
      146 +    resp = requests.get(url, timeout=20, headers={"User-Agent": user_agent})                                                                                    
      147 +    resp.raise_for_status()                                                                                                                                     
      148 +                                                                                                                                                                
      149 +    img = Image.open(io.BytesIO(resp.content)).convert("RGB")                                                                                                   
      150 +    img.thumbnail((MAX_IMAGE_EDGE_PX, MAX_IMAGE_EDGE_PX), Image.LANCZOS)                                                                                        
      151 +

      155 +

      157 +    cache_path.write_bytes(jpeg_bytes)

      459 +                err_f.write(                                                                                                                                    
      460 +                    json.dumps(                                                                                                                                 
      461 +                        {"custom_id": custom_id, "parse_error": str(e), "raw": raw[:500]},                                                                      
      462 +                        ensure_ascii=False,                                                                                                                     
      463 +                    ) + "\n"                                                                                                                                    
      464 +                )                                                                                                                                               
      465 +                failed += 1                                                                                                                                     
      466 +                continue                                                                                                                                        
      467 +                                                                                                                                                                
      468 +            out_row = dict(original.get(custom_id, {}))                                                                                                         
      469 +            out_row.update({                                                                                                                                    
      470 +                "vlm_label": annotation.get("vlm_label"),

      472 +                "vlm_confidence": annotation.get("vlm_confidence"),

      475 +                "vlm_finish_reason": finish_reason,                                                                                                             
      476 +            })

      478 +                out_row["vlm_raw_response"] = raw                                                                                                               
      479 +            out_f.write(json.dumps(out_row, ensure_ascii=False) + "\n")                                                                                         
      480 +            succeeded += 1                                                                                                                                      
      481 +                                                                                                                                                                
      482 +    if failed == 0:                                                                                                                                             
      483 +        err_path.unlink(missing_ok=True)
